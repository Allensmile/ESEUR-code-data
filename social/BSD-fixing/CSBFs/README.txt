Columns of the common-history and linked CSBFs:

1) T1 if it is the source change, T2 if it is the target change
2) cosine between commit notes
3) file name
4) date and time
5) system
6) committer id
7 file revision
8) YES if a common committer was found (i.e., the other committer is mentioned in the commit note or vice versa)
9) INTERESTING if the commit mentions the other system
10) list of contributors mentioned in the commit note
11) commit note
